# GlobalMart Sales Data Analysis

The growing availability of retail transaction data presents an opportunity for businesses to gain insights into sales performance, customer behaviour, and profitability. This project analyses GlobalMart sales data to identify key trends, evaluate product performance, and uncover factors affecting revenue and profit. The analysis supports data-driven decision-making by highlighting patterns across regions, product categories, and discount strategies.

---

## Overview

This end-to-end data analytics project performs exploratory data analysis (EDA) on GlobalMart sales data to examine relationships between sales, profit, product categories, and regional performance. The workflow includes data loading, cleaning, statistical analysis, and visualisation to extract meaningful business insights.

---

## Method Overview

The analysis follows a structured data analytics workflow:

- Data loading and inspection
- Data cleaning and preprocessing
- Exploratory data analysis
- Statistical analysis
- Data visualisation
- Insight generation

The dataset is first examined for structure and quality, followed by cleaning and preparation. Exploratory analysis is then conducted to identify patterns in sales and profitability across different segments.

---

## Project Workflow

### Data Loading

- Imported dataset using Pandas
- Inspected dataset structure and summary statistics
- Checked data types and column names

### Data Cleaning and Preparation

- Checked for missing values
- Verified duplicates
- Converted data types where necessary
- Prepared dataset for analysis

### Exploratory Data Analysis (EDA)

The analysis addresses key business questions, including:

- Overall sales performance
- Profitability trends
- Regional performance comparison
- Product category analysis
- Discount impact on profit
- Customer segment analysis

Visualisations include bar charts, histograms, scatter plots, and boxplots to highlight trends and relationships.

### Statistical Analysis

- Calculated summary statistics
- Compared mean sales and profit
- Analysed discount-profit relationships
- Identified high-performing categories

## Dataset

The GlobalMart dataset contains retail transaction data including:

- Order ID
- Order Date
- Ship Date
- Customer Segment
- Region
- Product Category
- Sub-category
- Sales
- Profit
- Discount

These variables are used to analyse business performance and profitability drivers.

## Results

The analysis revealed several key findings:

- Sales performance varies across regions
- Some product categories generate high revenue but lower profit
- Discounts significantly impact profitability
- Certain sub-categories contribute disproportionately to overall sales
- Profit distribution contains outliers affecting overall performance
